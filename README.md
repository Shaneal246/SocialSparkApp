## Description

The social app spark was created to act as a personal social assistance for users who want to maintain more active and meaningful relationship but may not know where to start.

## How to use 
.Enter time of the day 
Locate the input field at the top of the screen.Type in the keyword related to your current time or activity.

.Supported keywords: morning,mid -morning,afternoon,afternoon snack time,dinner,night,or after dinner.
.Tip: The app is smart! it doesn't matter if you use capital letters or accidentally add a space.

.Click the submit button .The app will instantly process your input and display a personalized social task in the result area.
 .For example, if you type "morning", it might suggest: : " Send a 'Good morning' text to family member!"

 .If you want to try a different time or start over, simply click the CLEAR button . This will wipe the screen clean so you can enter a new time of the day.
